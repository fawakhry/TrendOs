Matbagy Egypt V1856 Patch 11 - Accounting By Department

التحديث مبني فوق Patch 10 بدون حذف وظائف البرنامج الأساسية.

الجديد:
1) زر 💰 حسابات مطبعجي داخل شاشة الموظف.
2) ضياء:
   - يرى حسابات كل الأقسام.
   - يضيف خامات الليزر منفصلة عن خامات الطباعة.
   - يضيف البنود الثابتة: لامينشن، رولات طباعة 30/50/60، حبر بلوتر، بنود مشتركة.
   - يرى ربح/خسارة كل قسم.
3) وائل:
   - يسجل فاتورة قسم الطباعة فقط.
   - تظهر له خامات الطباعة والمشترك فقط.
4) جابر:
   - يسجل فاتورة قسم الليزر فقط.
   - تظهر له خامات الليزر والمشترك فقط.
5) رحمه / ريفان:
   - تقفيل الفاتورة النهائية.
   - تكتب الفاتورة كاملة يدويًا أو تستدعي أجزاء وائل وجابر حسب رقم الأوردر.
6) البنود المشتركة مثل الماكيت والسنيور:
   - كل قسم يسجل جزءه الخاص.
   - الفاتورة النهائية تتجمع عند رحمه/ريفان/ضياء.
