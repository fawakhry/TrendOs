# Matbagy Egypt V1856 Patch 12 - Clickable Stats Tabs

## الهدف
تحويل عدادات أعلى جدول الأوردرات إلى تابات قابلة للضغط. عند الضغط على أي عداد يتم فتح بياناته مباشرة في الجدول.

## التابات المضافة
- المعروض
- عاجل
- عادي
- متأخر
- مديونية
- مكبس
- استلامات بكرة
- غير مقفل لبكرة
- طباعة على الطاير
- ملغى
- مشاكل/متوقف

## طريقة العمل
كل زر من عدادات الإحصائيات أصبح `button` وله `data-stat-filter`.
عند الضغط يتم ضبط الفلاتر المناسبة تلقائياً:
- متأخر → فلتر متأخر
- مديونية → فلتر المديونيات
- مكبس → فلتر المكبس فقط
- استلامات بكرة → فلتر استلامات بكرة
- غير مقفل لبكرة → فلتر الأوردرات المطلوب تقفيلها لبكرة
- طباعة على الطاير → فلتر طباعة على الطاير
- ملغى → فلتر حالة ملغى
- مشاكل/متوقف → فلتر مشكلة/متوقف
