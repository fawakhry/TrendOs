TrendOS Web V1.3 - Customer Autocomplete + Operations Polish
============================================================

التعديلات:
1) عند إضافة أوردر، كتابة أول حرف من اسم الشات أو الرقم تعرض العملاء المتشابهين من شيت العملاء.
2) اختيار العميل يملأ الاسم + رقم العميل + نوع العميل تلقائيًا.
3) إظهار رقم العميل في كل واجهات التشغيل لوائل وجابر وباقي الأقسام.
4) إضافة بحث داخل جدول التشغيل برقم الأوردر / العميل / الرقم / البند.
5) إضافة فلاتر للحالة والأولوية.
6) ترتيب البنود: عاجل أولاً، ثم حسب الحالة.
7) عدادات سريعة للحالات.
8) إجبار المستخدم الذي ما زال على 0000 على تغيير كلمة المرور.
9) خدمة العملاء/admin فقط يستطيعون إضافة أوردر.
10) الموظفون لا يلمسون الشيت؛ التعديل فقط من الواجهة.

ملفات GitHub:
- index.html
- app.js
- styles.css
- config.js

ملف Apps Script:
- google_apps_script_trendos_web_v1_3.gs

طريقة الرفع:
1) على GitHub استبدل index.html و app.js و styles.css.
2) ارفع config.js فقط لو رابط apiUrl عندك نفس الرابط الموجود بالملف أو عدله قبل الرفع.
3) في Apps Script استبدل الكود بـ google_apps_script_trendos_web_v1_3.gs.
4) Save.
5) Deploy -> Manage deployments -> Edit -> New version -> Deploy.
6) افتح:
https://fawakhry.github.io/TrendOs/?v=1301
