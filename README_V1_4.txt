TrendOS Web V1.4

التحديثات:
1) ترتيب الأوردرات في كل واجهة:
   - العاجل أولاً
   - ثم الأقدم إلى الأحدث حسب ترتيب الصف في الشيت.

2) إظهار رقم الأوردر الحالي أعلى كل واجهة:
   - الطباعة
   - الليزر
   - خدمة العملاء / رحمة
   - الإدارة

3) إضافة قسم "متعدد الأقسام":
   عند اختيار هذا القسم من فورم إضافة الأوردر يتم إنشاء بندين تلقائيًا:
   - بند طباعة لمسؤول وائل
   - بند ليزر لمسؤول جابر
   بنفس رقم الأوردر وبـ Line ID مختلف:
   ORDER-01
   ORDER-02

4) باقي خصائص V1.3 مستمرة:
   - اقتراح العملاء بمجرد كتابة أول حرف
   - رقم العميل الخارجي ظاهر
   - بحث وفلاتر
   - تغيير كلمة المرور
   - صلاحيات الأقسام

الرفع:
1) على GitHub استبدل:
   index.html
   app.js
   styles.css
   config.js

2) في Apps Script استبدل الكود بملف:
   google_apps_script_trendos_web_v1_4.gs

3) Save

4) Deploy -> Manage deployments -> Edit -> New version -> Deploy
