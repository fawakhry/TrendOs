تعليمات تركيب TrendOS WhatsApp AI V1811

1) على GitHub ارفع الملفات التالية مكان القديمة:
   - app.js
   - index.html
   - styles.css
   - config.js

2) في Google Apps Script:
   - امسح الكود الحالي كله.
   - الصق محتوى ملف TrendOS_GOOGLE_APPS_SCRIPT_FULL_V1811_WHATSAPP_AI.gs.
   - اضغط Save.

3) اعمل نشر جديد:
   Deploy > Manage deployments > Edit > New version > Deploy

4) افتح البرنامج من GitHub Pages واعمل Ctrl + F5.

النتيجة:
- في كل أوردر هيظهر زر: AI يرد بالحالة.
- عند الحالات الجاهزة هيظهر زر إبلاغ العميل.
- عند الضغط يفتح WhatsApp برسالة جاهزة.
- بعد تأكيد الإرسال يتسجل في الشيت: آخر رسالة واتساب / آخر وقت / بواسطة / تم إبلاغ العميل.

ملاحظات:
- الإرسال هنا يدوي آمن؛ الموظف يضغط إرسال داخل واتساب.
- الإرسال التلقائي بالكامل يحتاج WhatsApp Cloud API و Templates معتمدة.
