TrendOS V1821 - متابعة اليوم + المتأخر + سجل الحركة

الملفات المطلوبة للرفع على GitHub مكان القديم:
- app.js
- index.html
- styles.css
- config.js

كود Google Apps Script:
- TrendOS_GOOGLE_APPS_SCRIPT_FULL_V1821_DASHBOARD_OVERDUE_LOG.gs

خطوات التركيب:
1) افتح GitHub repo.
2) ارفع app.js و index.html و styles.css و config.js مكان القديم.
3) اعمل Commit changes.
4) افتح Apps Script.
5) امسح الكود الحالي كله.
6) انسخ محتوى ملف TrendOS_GOOGLE_APPS_SCRIPT_FULL_V1821_DASHBOARD_OVERDUE_LOG.gs كله.
7) اضغط Save.
8) اعمل Deploy > Manage deployments > Edit > New version > Deploy.
9) افتح البرنامج واضغط Ctrl + F5.

الجديد:
- كارت متابعة اليوم للإدارة وخدمة العملاء.
- عداد متأخر حسب تاريخ التسليم المتوقع.
- علامة متأخر تظهر على الأوردر في الشاشة.
- سجل حركة الأوردرات يتعمل تلقائياً عند تسجيل أوردر، تغيير حالة/ملاحظات، أو تسجيل رسالة واتساب.
- نفس تعديلات V1820 موجودة: مكرر، إخفاء الجاهز/المسلم/المكرر، مكبس حراري، أولوية عادي افتراضيًا.
