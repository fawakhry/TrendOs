TrendOS Web V1.5 - Bug Fixes
============================

الإصلاحات:
1) حل خطأ:
   renderCurrentOrder is not defined

2) رحمة / خدمة العملاء يمكنها تسجيل أوردر بأقل بيانات:
   - اسم الشات فقط
   - القسم فقط
   وباقي الخانات اختيارية.

3) إذا فشل التسجيل يظهر لها بوضوح:
   لم يتم تسجيل الأوردر: سبب الخطأ

4) إذا تم التسجيل يظهر:
   تم تسجيل الأوردر بنجاح + رقم الأوردر + عدد البنود.

5) لو اسم البند فاضي، يتم تسجيله تلقائيًا:
   أوردر جديد - اسم القسم

6) قسم "متعدد الأقسام" يعمل:
   - بند طباعة لوائل
   - بند ليزر لجابر
   بنفس رقم الأوردر.

7) ترتيب الأوردرات:
   - عاجل أولاً
   - ثم الأقدم إلى الأحدث.

8) رقم الأوردر الحالي ظاهر أعلى كل واجهة.

1) على GitHub استبدل:
   index.html
   app.js
   styles.css
   config.js

   google_apps_script_trendos_web_v1_5.gs
